🔧 TROUBLESHOOTING
═════════════════════

❌ "Cannot find module 'express'"
→ npm install
→ Confirmă că package.json e corect

❌ "OPENAI_API_KEY is not defined"
→ Verifică .env
→ Confirmă că e salvat

❌ "Port 3000 in use"
→ Schimbă PORT în .env (3001, 3002, etc.)
→ SAU: kill -9 $(lsof -t -i:3000) [Mac/Linux]

❌ "401 Unauthorized"
→ API Key invalid
→ Mergi pe: platform.openai.com/api-keys
→ Generează o nouă cheie

❌ "429 Too Many Requests"
→ Rate limit atins
→ Așteaptă și reîncearcă

❌ "Frontend se conectează dar e offline"
→ Verifică că serverul rulează: npm start
→ Verifică console (F12) pentru erori
