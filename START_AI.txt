⚡ VERSIUNE RAPIDĂ (5 MINUTE)

   → Copie cheia (sk-proj-...)

2️⃣ Setează cheia:
   → Deschide: .env
   → Schimbă linia 1:
   OPENAI_API_KEY=sk-proj-YOUR_KEY_HERE
   
3️⃣ Instalează backend:
   Deschide PowerShell în folder și rulează:
   npm install

4️⃣ Test:
   npm test
   (Trebuie să apară "✅ SUCCES!")

5️⃣ Pornire:
   npm start
   (Lasă fereastra deschisă!)

6️⃣ Verifica:
   - Deschide index.html
   - Mergi la "Asistent AI"
   - Pune o întrebare
   - Ar trebui să ai răspuns! 🎉

❌ PROBLEME?
═════════════

"npm: command not found"
→ Instalează Node.js de pe nodejs.org

"npm install fail"
→ Rulează: npm install --no-cache

"OPENAI_API_KEY invalid"
→ Copiază din nou cheia exact

"Port 3000 e ocupat"
→ Schimbă PORT în .env la 3001

"Frontend nu se conectează"
→ Confirmă că npm start rulează
