## ЁЯЪА What You Need To Do

### IMMEDIATE (Today):

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Setup MySQL Database:**
   ```bash
   mysql -u root -p
   source database.sql
   ```

3. **Update `.env`:**
   ```env
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=your_password
   DB_NAME=fixmind
   JWT_SECRET=change-this-in-production
   ```

4. **Replace server.js:**
   ```bash
   mv server.js server-old.js
   mv server-new.js server.js
   ```

5. **Test Locally:**
   ```bash
   npm start
   # Visit: http://localhost:3000/auth.html
   ```

### Testing Flow:
1. Open `http://localhost:3000/auth.html`
2. Click "├Оnregistrare" (Register)
3. Select "Profesor"
4. Fill form & register
5. Should redirect to professor dashboard
6. Create a class
7. Copy invitation code
8. In new browser/incognito: Register as "Elev"
9. Join class with code
10. Should appear in professor's dashboard

## ЁЯУК Database Structure

**users** - Core user data
- id, email, password_hash, first_name, last_name, type (profesor/elev)
- oauth fields for future social login

**profesori** - Professor info
- id, user_id, materia, biografie, scoala

**clase** - Classes
- id, profesor_id, nume (ex: Clasa IX A), cod_invitatie, descriere

**inscrisuri** - Student enrollment
- id, elev_id, clasa_id, progres_procent
- Ensures one-to-one student-class relationship

**teste** - Tests
- id, clasa_id, profesor_id, titlu, descriere

**intrebari** - Questions
- id, test_id, tip (choice/short/essay), text, puncte

**rezultate** - Scores
- id, elev_id, test_id, puncte_obtinute, procent_corect, data
