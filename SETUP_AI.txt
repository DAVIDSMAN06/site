ARHITECTURA:
  Frontend (HTML/JS) ←→ Backend (Node.js + Express) ←→ OpenAI API

FLUXUL:
  1. Utilizator pune întrebare în chat
  2. Frontend trimite la server local
  3. Server validează că e despre materie
  4. Server apelează OpenAI API
  5. OpenAI răspunde
  6. Server trimite răspunsul la frontend
  7. Răspunsul apare în chat

───────────────────────────────────────────────────────────────────

PASUL 1️⃣: SETUP NODE.JS
═══════════════════════

1. Descarcă Node.js
   → Mergi pe: https://nodejs.org/
   → Descarcă versiunea LTS (Long Term Support)
   → Instalează-l pe calculatorul tău

2. Verifică instalația
   Deschide PowerShell/CMD și tastă:
   
   node --version
   npm --version
   
   (Trebuie să afișeze versiunile - confirmă că e instalat)

───────────────────────────────────────────────────────────────────

PASUL 2️⃣: OBȚINEȚI API KEY DE LA OPENAI
══════════════════════════════════════════

1. Mergi pe: https://platform.openai.com/api-keys

2. Conectează-te cu contul tău (sau creează unul)

3. Apasă "Create new secret key"

4. Copie cheia (va arăta ceva de genul: 
   sk-proj-ABC123XYZ...)

5. PAZĂ-O! Nu o posta nicăieri public! 🔐

(Notă: OpenAI oferă $5 credite gratuite pentru noi utilizatori)

───────────────────────────────────────────────────────────────────

PASUL 3️⃣: SETUP BACKEND (NODE.JS)
═════════════════════════════════

1. Mergi în folderul proiectului
   
   cd C:\Users\DAVIDSMEN\Desktop\sitebun1

2. Instalează dependențele
   
   npm install
   
   (Aceasta va descărca: express, axios, cors, dotenv)

3. Creează fișierul .env (de la .env.example)
   
   Deschide .env.example și:
   - Copiază conținutul într-un nou fișier
   - Numește-l: .env (fără extension)
   - Înlocuiește: your_openai_api_key_here
            cu cheia ta din pasul 2
   
   Exemplu:
   OPENAI_API_KEY=sk-proj-ABC123XYZ...
   PORT=3000
   NODE_ENV=development

4. Salvează fișierul .env în rădăcina proiectului

───────────────────────────────────────────────────────────────────

PASUL 4️⃣: PORNIRE BACKEND
════════════════════════════

1. Deschide PowerShell în folderul proiectului

2. Tastă:
   
   npm start
   
   (Ar trebui să vezi: 🤖 AI Server funcționează pe http://localhost:3000)

3. LASĂ FEREASTRA DESCHISĂ!
   (Serverul trebuie să ruleze ca să funcționeze AI-ul)

───────────────────────────────────────────────────────────────────

PASUL 5️⃣: TESTEAZĂ AI-UL
══════════════════════════

1. Cu serverul pornit, deschide: index.html în browser

2. Mergi la: "Asistent AI"

3. Ar trebui să vezi mesajul: "🤖 AI Real Conectat!"

4. Pune-i o întrebare despre gândire:
   "Ce este gândirea?"
   
   Ar trebui să primești un răspuns de la OpenAI! ✅

───────────────────────────────────────────────────────────────────

🔧 TROUBLESHOOTING
═════════════════════

❌ EROARE: "Backend nu e pornit"
→ Soluție: Asigură-te că npm start rulează în PowerShell

❌ EROARE: "API Key invalid"
→ Soluție: Verifică .env - cheia trebuie să fie corectă

❌ EROARE: "Port 3000 e deja în uz"
→ Soluție: Schimbă PORT în .env la altul (3001, 3002, etc.)

❌ EROARE: npm install nu funcționează
→ Soluție: Asigură-te că ai Node.js instalat corect

❌ AI-ul răspunde la orice întrebare
→ Soluție: Contextul e bine setat - ar trebui să refuze alte subiecte
