HOW IT WORKS - QUICK OVERVIEW
═════════════════════════════════════════════════════════════════════════════

USER ASKS QUESTION → FRONTEND (JavaScript)
         ↓
FRONTEND SENDS → HTTP POST to localhost:3000/api/chat
         ↓
BACKEND RECEIVES → Validates domain (Is it about course?)
         ↓
IF VALID → Calls OpenAI API with system prompt
IF INVALID → Returns refusal message (costs $0!)
         ↓
OPENAI RESPONDS → Real AI answer (1-3 seconds)
         ↓
BACKEND RETURNS → Response to frontend
         ↓
FRONTEND DISPLAYS → Beautiful chat message
         ↓
USER SEES ANSWER ✅
