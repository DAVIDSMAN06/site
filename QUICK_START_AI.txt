⏱️ RAPID START
══════════════

1. Descarcă Node.js: https://nodejs.org/

2. Obțineți API Key: https://platform.openai.com/api-keys

3. În folderul proiectului:
   
   npm install

4. Crează .env (copie din .env.example):
   
   OPENAI_API_KEY=sk-proj-YOUR_KEY_HERE

5. Pornire server:
   
   npm start

6. Deschide asistent.html și pune întrebări! 🤖

❓ FAQ
═══════

P: De ce are nevoie de backend? Nu putem folosi OpenAI direct din JS?
R: NU - API Key-ul ar fi expus. Backend-ul o ține privată și validează.

P: Costă mult OpenAI?
R: Puțin! ~0.001-0.01$ pe întrebare. Novices primesc $5 gratuit.

P: Funcționează offline?
R: Parțial - mod local cu răspunsuri preimplementate.

P: Cum pot vedea costo cheltuielilor?
R: https://platform.openai.com/account/usage

P: Pot seta un limit de cheltuieli?
R: DA - Settings → Billing → Usage limits

🔥 QUICK DEBUG
═══════════════

Problem: npm install da eroare
Fix: Controlează dacă Node.js e în PATH
  node --version

Problem: npm start nu pornește
Fix: Asigură-te că .env e creat corect

Problem: AI spune "Backend nu pornit"
Fix: Controlează dacă npm start funcționează
  Poate portul 3000 e ocupat - schimbă PORT în .env

Problem: AI răspunde "Eroare API"
Fix: Verifică cheia OpenAI în .env
  Ar trebui: OPENAI_API_KEY=sk-proj-...

Problem: AI refuză tot
Fix: Asta e corect! E restrictiv la materie.
  Pune o întrebare despre gândire.
