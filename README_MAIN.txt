🐛 TROUBLESHOOTING
════════════════════

❌ "npm: command not found"
   → Reinstalează Node.js

❌ "ENOTFOUND openai"
   → Check internet
   → OpenAI server e down?

❌ "401 Unauthorized"
   → API Key e invalid
   → Generează nouă cheie

❌ "Port 3000 occupied"
   → Schimbă PORT în .env

❌ "Cannot find module"
   → npm install nu a mers
   → Rulează din nou
