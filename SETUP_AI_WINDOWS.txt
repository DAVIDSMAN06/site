PASUL 1: DESCARCĂ ȘI INSTALEAZĂ NODE.JS
═════════════════════════════════════════

1. Mergi pe: https://nodejs.org/

2. Descarcă versiunea LTS (Recommended)
   Ar trebui să fie undeva ca: v18.x.x LTS

3. Rulează instalatorul .msi

4. Follow wizard default (Next, Next, Install)

5. Verifică că e instalat:
   
   Deschide PowerShell (Win+X → Windows PowerShell)
   Tastă:
   
   node --version
   npm --version
   
   Trebuie să arată versiuni (ex: v18.16.0, 9.6.7)

───────────────────────────────────────────────────────────────────

PASUL 2: OBȚINEȚI API KEY DE LA OPENAI
═══════════════════════════════════════

1. Deschide browser și mergi pe:
   https://platform.openai.com/api-keys

2. Conectează-te / Creează cont

3. Apasă: "Create new secret key"

4. Copie cheia (va arăta: sk-proj-ABC123XYZ...)

5. PAZĂ! O vei folosi din nou.

PASUL 3: SETUP BACKEND
══════════════════════

1. Deschide PowerShell și mergi în folder:
   
   cd C:\Users\DAVIDSMEN\Desktop\sitebun1

2. Instalează dependențe:
   
   npm install
   
   (Acceptă warning-uri, ar trebui să se termine cu "added X packages")

3. Deschide .env.example cu Notepad/VSCode
   
   Conținut:
   ────────────────────────────
   OPENAI_API_KEY=your_openai_api_key_here
   PORT=3000
   NODE_ENV=development
   ────────────────────────────

4. Crează un NOU fișier (File → New) și salvează ca ".env"
   
   Copiază conținutul din .env.example

5. Schimbă rândul 1:
   
   OPENAI_API_KEY=sk-proj-ABC123XYZ...
   
   (Înlocuiește cu cheia ta din pasul 2!)

6. Salvează fișierul în rădăcina proiectului

🆘 PROBLEME ȘI SOLUȚII
════════════════════════

PROBLEMA 1: "npm: command not found"
→ Node.js nu e în PATH
→ SOLUȚIE: Restartează calculatorul după install Node.js

PROBLEMA 2: "Cannot find module 'express'"
→ npm install nu a rulat
→ SOLUȚIE: Rulează: npm install

PROBLEMA 3: Port 3000 e deja ocupat
→ Alt program folosește portul
→ SOLUȚIE: Schimbă PORT în .env (3001 de ex.)
→ SOLUȚIE ALT: PowerShell → taskkill /IM node.exe

PROBLEMA 4: "OPENAI_API_KEY is not defined"
→ .env nu e setat corect
→ SOLUȚIE: Creează .env (NU .env.example!)
→ SOLUȚIE: Verifică că .env e în rădăcina proiectului

PROBLEMA 5: API Key "401 Unauthorized"
→ Cheia e invalidă/expirată
→ SOLUȚIE: Generează o nouă cheie pe platform.openai.com

PROBLEMA 6: "Frontend nu se conectează la server"
→ Serverul nu rulează
→ SOLUȚIE: Confirmă că npm start lucra și portul e corect
